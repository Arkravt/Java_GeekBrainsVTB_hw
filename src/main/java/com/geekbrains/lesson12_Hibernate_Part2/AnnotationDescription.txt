1. @Entity - Указывает на то, что мы хотим сохрнаять наши объекты в БД.

2. @Table - Указываем таблицу в который эта сущность будет храниться.
Атрибуты:
    name - имя таблицы в БД.
    indexes - можем на таблицу повесить индексы. По одному полю, двум и т.д.

3. @Column - Указывает, что это поле будет использоваться в качестве колонки в БД.
Атрибуты:
    name - имя таблицы в БД.
    nullable - возмоность установить проверку на null.
    length - указать длину строки. По умолчанию 255.
    updatable - возможность изменять поле.
    columnDefinition - позволяет для столбца создать своё описание.

4. @org.hibernate.annotations.ColumnTransformer - модифицирует записи при чтение/записи в бД по формуле.

5. @org.hibernate.annotations.Formula - Возможность указывать свою формулу.

6. @Temporal - Хранит дату/время/датуВремя.

7. @org.hibernate.annotations.CreationTimestamp - в момент первого сохранения объекта в базе должен время сохранить.
