1. <bean> - сущность.
    id - имя класса которму будем получить бин
    class - путь к классу
        1.2. <constructor-arg> - Внедрение зависимостей с помощью контструктора
            ref - указывается ссылочное свойство конструктора
            value -указывается примитивное сваойство конструктора
            index - порядковый нмоер свойства в конструкторе
            type - тип свойства в конструкторе
        1.3. <property> - Внедрение зависимостей с помощью сеттеров
            name - указывается название сеттера без префикса set
            ref - указывается ссылка на бин
            value - указывается значение
            * С помощью "p:" можно(не обязательно) сделать запись короче. См. пример.

2. В файле car.properties хранятся значения которые подставляются
по ключу в файл lesson13_example_Context.xml. Пример ${car.class}.
Файл car.properties создавать не обязательно.
